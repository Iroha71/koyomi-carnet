# こよみのカルネ

## 概要

日々の支出を記録し、予算管理や支出の把握を行えるAndroidアプリ。
ユーザーの使える資金の把握や物品を購入する際の判断を行える状態にする。

## 機能

- 予算設定
  - 1か月の予算を設定する
  - 設定は引き継ぐことができる
- 支出記録
  - その日の支出を記録する
  - 1日の時間帯ごとに支出を記録できる
  - 支出にはカテゴリを設定できる
- 固定費設定
  - 予め固定費を設定できる

## 画面構成

- ホーム画面（ダッシュボード）
  - 当月の予算・支出合計・残額を表示する
  - 固定費を差し引いた使える資金を把握できる
  - 当日や直近の支出を確認できる
  - 各機能（支出記録・予算設定・固定費設定）への導線を持つ
- 支出記録画面
  - その日の支出を入力・登録する
  - 1日の時間帯（朝・昼・夜など）ごとに支出を記録する
  - 支出にカテゴリを設定する
  - 金額・メモを入力する
- 支出一覧（履歴）画面
  - 記録した支出を日付・時間帯ごとに一覧表示する
  - カテゴリ別の支出を確認できる
  - 各支出の編集・削除を行える
- 予算設定画面
  - 1か月の予算を設定・変更する
  - 前月の設定を引き継ぐ
- 固定費設定画面
  - 固定費の項目・金額を登録・編集・削除する
  - 登録した固定費の一覧を確認できる
- カテゴリ設定画面
  - 支出に紐づけるカテゴリを追加・編集・削除する
